remove :: Char -> String -> String Source #

Remove first occurrence of a character from the word.

c.f. http://hackage.haskell.org/package/base/docs/Data-List.html#v:delete

isValid :: String -> String -> Bool Source #

Check if a word contains only characters from a list. If all valid characters are removed from the word, and there are still characters left over, then the word is not valid. If all valid characters are removed from the word, and the word is empty, then the word is valid.

See also Data.List intersection function, \\

isPlural :: String -> Bool Source #

Check if word is a plural.

This assumes the word is at least 2 characters long.

  1. False if word does not end in s
  2. False if word ends in "ss"
  3. True Otherwise as ends in single s
